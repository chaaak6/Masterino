# Keyboard Shortcuts

Masterion provides a comprehensive set of keyboard shortcuts so you can work without constantly reaching for the mouse. Master a few key ones and your workflow noticeably speeds up ŌĆö especially when switching Agents, managing conversations, or navigating the interface.

## How to Access Shortcut Settings

Click your user avatar in the top-right corner ŌåÆ **App Settings** ŌåÆ **Shortcuts**. You can view all shortcuts and customize most of them.

## Essential Shortcuts

| Action                             | Shortcut                               |
| ---------------------------------- | -------------------------------------- |
| **Open Command Menu**              | `Ōīś + K` (Mac) / `Ctrl + K` (Win/Linux) |
| **Focus search**                   | `Ōīś + J` / `Ctrl + J`                   |
| **Switch to pinned Agent**         | `Ctrl + 1ŌĆō9`                           |
| **Switch to default conversation** | `Ctrl + 0`                             |
| **Toggle left panel**              | `Ōīś + [` / `Ctrl + [`                   |
| **Toggle right panel**             | `Ōīś + ]` / `Ctrl + ]`                   |
| **Focus mode**                     | `Ōīś + \` / `Ctrl + \`                   |
| **Shortcut help**                  | `Ctrl + Shift + ?`                     |

**Focus mode** hides all panels except the current conversation ŌĆö ideal for distraction-free writing or long reads.

## Conversation Shortcuts

| Action                          | Shortcut                             |
| ------------------------------- | ------------------------------------ |
| **Open conversation settings**  | `Ōīś + ,` / `Ctrl + ,`                 |
| **Regenerate message**          | `Ōīś + R` / `Ctrl + R`                 |
| **Delete last message**         | `Ōīś + D` / `Ctrl + D`                 |
| **Delete and regenerate**       | `Ōīś + Shift + R` / `Ctrl + Shift + R` |
| **New Topic**                   | `Ōīś + N` / `Ctrl + N`                 |
| **Add message without sending** | `Ōīś + Enter` / `Ctrl + Enter`         |
| **Edit message**                | `Ctrl + Alt` + double-click message  |

**Add message without sending** ŌĆö useful when you want to add context to the conversation without triggering an immediate response. The Agent will see the new message when it next replies.

## Custom Shortcuts

Most shortcuts can be customized. In the Shortcuts settings panel, click the shortcut you want to change, then press your desired key combination. Useful when:

- Your muscle memory conflicts with another app
- You want to prioritize actions you use most
- You're on a different keyboard layout

**Tip:** Start with the Command Menu (`Ōīś + K` / `Ctrl + K`) ŌĆö it's the fastest way to jump to any Agent, Topic, or setting.

## Use Cases

**Power-user workflow** ŌĆö `Ōīś + K` to open Command Menu ŌåÆ type Agent name ŌåÆ Enter. Switch Agents in under a second.

**Quick iteration** ŌĆö `Ōīś + R` to regenerate. `Ōīś + Shift + R` if you want to delete the last message first, then regenerate.

**Distraction-free writing** ŌĆö `Ōīś + \` for Focus mode. No sidebar, no panels ŌĆö just the conversation.

**Multi-Agent work** ŌĆö Pin your top Agents to the sidebar, then use `Ctrl + 1ŌĆō9` to switch between them without touching the mouse.
